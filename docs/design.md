高性能ID生成器

基于雪花算法的优化版本，支持：
- 高并发生成
- 时钟回拨处理
- 性能监控
- 批量生成
- ID预分配池
- 本地缓存

高性能架构
- ID预分配池：减少计算开销
- 本地缓存：支持业务标签缓存
- 无锁设计：使用原子操作和CAS
- 批量生成：减少函数调用开销

容错机制
- 时钟回拨检测与处理
- 序列号溢出处理
- 节点故障转移

监控体系
- Prometheus指标收集
- 实时性能监控
- 业务统计

📊 性能指标
- QPS: 100万+ (通过ID池和缓存优化)
- 延迟: < 1ms (本地内存操作)
- 可用性: 99.99% (容错机制)
- 扩展性: 支持1024个节点，32个数据中心

🛠️ 技术栈
- 语言: Rust (高性能，内存安全)
- 框架: Axum (异步HTTP框架)
- 缓存: DashMap (高性能并发哈希表)
- 监控: Prometheus + Grafana
- 部署: Docker + Kubernetes
